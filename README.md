# FlowerOS

*A small learning OS*

## Building

You'll need to install `nasm`, `ld`, `qemu` (to run in virtual machine),
`grub`, and `make`.

You can build it with `make build`, make the iso with `make iso`, and
launch qemu and run it with `make run`.
